# Student Management System

A robust, console-based Student Management System built with Python and Object-Oriented Programming (OOP) principles. Designed to handle CRUD operations for students, faculty, courses, and enrollments using a relational database schema.

## Features
* **Secure Authentication:** Role-based login system (Admin, Faculty, Student).
* **Environment Configuration:** Uses `.env` for secure database credential management.
* **Database Connectivity:** Configured for SQLite (local dev) with structural support for MySQL/PostgreSQL.
* **Modular OOP Architecture:** Clean separation of models, database connections, and application logic.

## Tech Stack
* **Language:** Python 3
* **Database:** SQLite / MySQL ready
* **Environment:** `python-dotenv`

## Installation
1. Clone the repository: `git clone https://github.com/jeevankurapati07-stack/StudentManagementSystem.git`
2. Install dependencies: `pip install python-dotenv`
3. Configure your `.env` file based on the provided placeholders.
4. Run the application: `python main.py`
